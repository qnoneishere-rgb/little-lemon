# Little Lemon Capstone Project
This is a React-based table booking application for the Little Lemon restaurant.

## Features
- Interactive booking form.
- Form validation for guests and dates.
- Responsive design for mobile and desktop.

## Setup Instructions
1. Clone the repository.
2. Run `npm install`.
3. Run `npm start` to view the app.
4. Run `npm test` to run the unit tests.
